# -Subarray-Sum-Equals-K
Efficient C++ solution for LeetCode 560 - Subarray Sum Equals K using the Prefix Sum and Hash Map approach. The solution counts the number of continuous subarrays whose sum equals a given target value in O(n) time complexity.

# Subarray Sum Equals K

## Problem Statement
Given an integer array `nums` and an integer `k`, return the total number of continuous subarrays whose sum equals `k`.

### Example 1
Input:
nums = [1,1,1]
k = 2

Output:
2

### Example 2
Input:
nums = [1,2,3]
k = 3

Output:
2
